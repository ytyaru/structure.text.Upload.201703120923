# このソフトウェアについて

構造化テキストデータをアップロードする場。

HTMLなどから参照するtsv,csv,ini,xml,json,jsonldなどの構造化テキストデータをアップロードする。

d3.jsの`d3.dsv()`, `d3.csv()`, `d3.text()`などから参照するファイルを配置する予定。
